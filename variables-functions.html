Función var()

Una custom property o propiedad personalizada es, una vartiable, es decir un espacio reservado en la computadora para almacenar datos. Tienen un alcance o scope en el lugar donde esten definidas, que en el caso de CSS es el selector en el que definamos la declaración de la variable y todos sus elementos subsecuentes. Las variables se tienen que definir dentro de un selector

Para definir una variable, hacemos uso de la palabra reservada var y dos paréntesis. Es una función var(), dentro de estos llamamos la variable que queremos usar.

Las variables solo existen para los elementos hijos en el ámbito que esta ha sido definida, si queremos usarla fuera de este, no funciona, pero recibe un segundo parámetro que si se aplicará al elemento.

Para que las variables puedan existir en todos los elementos HTML, entonces deben ser declaradas en la etiqueta html; usando una pseudoclase llamada root (:root). Esta hace referencia a la etiqueta html, pero tiene mayor peso específico.

Las variables también pueden ser definidas en la etiqueta style de HTML.

Función min() & max()

Funciones min() & max()

Las funciones min() & max() toma como parámetros dos o más valores y utiliza uno de ellos según las condiciones del viewport, cuando esta condición cambia toma el siguiente parámetro. La función mínimo tomará el menor parámetro, hasta que la circunstancia de la pantalla requiera que tome el mayor. La función maximo hará lo contrario.

Los valores siempre deben estar separados por comas y pueden ser múltiples.

Función clamp()

Toma en primer lugar el tamaño mínimo, después toma el tamaño ideal, y por último el tamaño máximo. Siendo un buen complemento para min(), max().
