Estilos de Fuentes

Font-family

Es el conjunto de tipografías que podemos aplicar, el navegador, por defecto usa un tipo de tipografía (Georgia, "Times new roman", Times, Serif).

El estilo de fuente varía de acuerdo al sistema operativo o la computadora que el usuario disponga.

Font-size

Define el tamaño de la fuente. Ya hemos visto anteriormente que podemos usar diversos tipos de unidad para definir el tamaño. El que el navegador lee por defecto es el pixel.

Font-weight

Esta propiedad define el grueso o peso del la fuente. 400 es el grosor regular de la tipografía.

100 es la más delgada o ligera, después puede expresarse con nombre como bold (con un peso de 700), o extra bold (900) por citar algunos. 900 Es la más gruesa o pesada.

El peso depende del diseño de la tipografía, no todas tienen el mismo grosor por defecto.

Font-style

Define el estilo que le vamos a dar, puede ser italc (cursiva), oblicua o normal.

Font-variant

permite hacer algunas conversiones, son variantes de la misma fuente, se puede convertir todo a mayúscula por ejemplo

Line-height

Esta propiedad define la altura que va a tener cada línea de nuestros párrafos, es decir, la altura entre líneas.

Font

Es una propiedad short hands, que permite ingresar los distintos estilos, cada estilo debe separar por un espacio en blanco, y deben tener cierto orden al ser definidos. No es práctico, pues hay que estar memorizando el orden.

Font-display

La propiedad font-display nos dice como queremos que nosd cargue el texto, mientras la tipografía personalizada se carga. Es pues un complemento del font.

auto: Permite que el navegador use el método predeterminado que suele ser font.
block: oculta brevemente el texto hasta que la fuente haya sido descargada por completo.
swap: indica al navegador que utilice la fuente alternativa para visualizar el texto, hasta que la fuente personalizada se haya cargado por completo.
falback: Es una mezcla de auto y swap.
optinal: ocultará el texto, luego lo cargará con la fuente alternativa, y finalmente aplicará la fuente personalizada.

Google Fonts

Google Fonts

Google Fonts es la página más famosa e utilizada para descargar fuentes tipográficas, textos e íconos.

Al seleccionar una tipografía, la página nos muestra todas las variantes, grosor, minúsculas, mayúsculas y estilos.

Al elegir una tipografía debemos tener en cuenta si sus caracteristicas estan de acuerdo a las exigencias del documento en que se van a aplicar (que contenga la ñ por ejemplo, si el texto es en español).

Cuántas tipográfias usar?

Ya usar tres tipografías o más se puede considerar una mal práctica, sobre todo si son externas, por que relentizaran la carga del documento a la hora de visualizar el contenido.

Usar un tipo de tipografía para los textos y otro para títulos, enlaces, etc está bien.

Es mejor ir variando el puntaje, tamaño, etc de un mismo tipo de letra.

Text Align

Nos alinea el texto, su valor por defecoto es hacia la izquierda (left), pero podemos alinearlo a la derecha (right), justificarlo (justify) o centrarlo (center).

Text Decoration

Nos permite varios valores como underline (subrayado por debajo), overline (la raya por arriba), color, etc. El valor por dedecto es none

Text Indent

Le da sangría al texto (un espaciado).

Text Transform

Esta propiedad sirve para capitalizar, puede poner la primer letra de cada palabra en mayúscula, todo en mayúscula, todo minúscula, etc.

Letter Spacing

Es el espaciado entre cada caracter del texto.

Text Overflow

Sirve para cortar el texto cuando desborda la caja y le pone puntos suspensivos, este valor se llama ellipsis, clip encambio, oculta el texto desbordado.

El overflow debe estar en modo hidden (oculto) y white-space con valor de nowrap (lleva todo a una línea).
